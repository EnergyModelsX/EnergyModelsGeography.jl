EnergyModelsGeography — Module

Exports:

source

The Julia package extends EnergyModelsBase.jl with functionality to set up an energy system consisting of several separate regions, with transmissions between.

The extension focuses on overriding the function EMB.create_model to add additional types, variables, and constraints to the model.
